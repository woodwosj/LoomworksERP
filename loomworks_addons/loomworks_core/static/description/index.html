Loomworks ERP Core

AI-First Open-Source ERP Branding & Configuration

Overview

Loomworks Core is the foundation module for Loomworks ERP, providing custom branding, color palette, and base configuration for the entire system.

Features

  • Custom Branding: Loomworks logos, favicon, and icons throughout the UI
  • Color Palette: Professional deep blue (#1e3a5f) and teal (#2dd4bf) theme
  • Page Titles: "Loomworks ERP" branding in browser tabs
  • Login Page: Custom branded login experience
  • SCSS Variables: Extensible theme system for child modules
